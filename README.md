# Visualization-with-Seaborn-and-Matplotlib
Titanic Data Visualization Project: This repository contains Python scripts using Seaborn and Matplotlib to explore the Titanic dataset. It includes various visualizations such as countplots, swarmplots, pie charts, heatmaps, violin plots, and scatter plots.
